# Small-Space Setup Lab feedback

This public repository is the correction channel for [Small-Space Setup Lab](https://small-space-setup-lab.pages.dev/).

If a measurement, compatibility note, price-freshness date, retailer destination, or evidence label is wrong, [open a correction](https://github.com/222-glitch/small-space-setup-lab-feedback/issues/new?template=correction.yml). Include the affected page and a manufacturer or retailer source when possible. Do not post private account, order, address, or payment information.

Corrections are reviewed against the site's evidence policy before the public catalog is updated.
